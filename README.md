# Brandly

> **Grow Your Brand With Smart Marketing**

Brandly is a growth-focused marketing agency specializing in personal branding, high-impact creative content, and data-driven performance marketing. This repository contains the official digital presence of the agency, built with a modern, high-performance tech stack.

---

## 🚀 Core Services

### 👤 Personal Branding
We build authoritative digital presences for founders, executives, and visionaries.
- LinkedIn Profile Optimization & Management
- Long-term Brand Strategy & Positioning
- Thought Leadership Content Production

### 🎨 Creative & Content
High-impact visual assets designed for the modern consumer.
- High-Conversion Ad Creatives (Static & Motion)
- Social Media Aesthetic & Content Planning
- Professional Brand Photoshoots & Video

### 📈 Performance Marketing
Precision-targeted campaigns designed to maximize ROI.
- Meta Ads (Instagram & Facebook) Management
- Google Search & Display Campaign Strategy
- Advanced Retargeting & Funnel Optimization

---

## ✨ Key Features

- **Responsive Design**: Seamless experience across mobile, tablet, and desktop.
- **Premium Aesthetics**: Curated color palettes (Cream, Primary Red, Charcoal) and modern typography (Outfit/Inter).
- **Smooth Animations**: High-performance micro-animations and transitions powered by Framer Motion.
- **Dark Mode Support**: Fully integrated dark theme for a modern user experience.
- **Interactive UI**: Dynamic components including case study carousels, service pillars, and contact forms.

---

## 🛠️ Technology Stack

- **Frontend**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Routing**: [React Router Dom v7](https://reactrouter.com/)
- **Backend/Database**: [Firebase](https://firebase.google.com/)
- **Icons**: [Material Symbols](https://fonts.google.com/icons)

## 🏁 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Dhwanish08/brandly.git
   cd brandly
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Production

Build the project for production:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```
